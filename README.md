# ⏰ Relógio Digital com Data e Saudação

Projeto de um relógio digital com data, saudação dinâmica (Bom dia, Boa tarde, Boa noite) e mudança de fundo conforme o horário.

## 🧠 Funcionalidades

- Exibe a hora atual (horas, minutos e segundos)
- Mostra a data atual (dia, mês e ano)
- Saudação automática baseada no horário:
  - 🌅 Bom dia (5h às 12h)
  - ☀️ Boa tarde (13h às 18h)
  - 🌙 Boa noite (demais horários)
- Troca automática do fundo da página entre tema claro (dia) e escuro (noite)

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript puro (Vanilla JS)

## 🎨 Layout

- Interface simples e intuitiva
- Fundo com gradiente colorido de acordo com o período do dia
- Relógio digital estilizado com bordas arredondadas e efeito de sombra
